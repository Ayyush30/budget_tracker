# 💰 Personal Budget Tracker

A full-stack web application to manage personal finances by tracking income and expenses.  
Built using HTML, CSS, JavaScript, PHP, and MySQL as part of an Agile-based 5-week project.

---

## 🚀 Features

### ✅ Week 1 – Planning
- Product backlog (user stories)
- UI mockups
- Technology stack selection

### ✅ Week 2 – Core Functionality
- Add transaction form
- Input validation (required fields, numeric amount)
- Store transactions in array
- Display transactions in table

### ✅ Week 3 – CRUD Operations
- ✏️ Edit transactions
- 🗑️ Delete transactions (with confirmation)
- 🔄 Real-time UI updates

### ✅ Week 4 – Advanced Features
- 🔍 Filter transactions (category & type)
- ↕️ Sort transactions (date & amount)
- 📊 Summary dashboard (income, expense, balance)
- 📈 Charts (Pie & Bar using Chart.js)

### ✅ Backend Integration
- PHP API (add, get, update, delete)
- MySQL database for persistent storage
- Fetch API for frontend-backend communication
